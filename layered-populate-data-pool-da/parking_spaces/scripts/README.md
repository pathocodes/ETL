# Parking Spaces Scripts

This folder `parking_spaces/scripts` contains the notebooks for data research, export, transformation, unification, and database upload.

## `parking_spaces_data_modelling.ipynb`

Tis notebook contains step 1 data modeling and research.

* Data source discovery

* Data modelling and planning

## `parking_spaces_data_transformation`

This notebook contains data transformation and preprocessing (step 2).

* Data integration and enrichment

* Verifying data consistency

* Final proposed schema and table justification

* Test upload into Neon database

* Preprocessing notes

## `parking_spaces_populating_db`

The last notebook contains the full ETL pipeline to extract the raw data from different sources, transform the data into a final cleaned unified dataset, and load the cleaned data into the Layered database.
